# thinkstem-legal

このリポジトリは、各アプリのプライバシーポリシーなど法務関連ページを公開するためのリポジトリです。

## 目的

- GitHub Pages で法務ページを公開する
- アプリ本体コード（private）とは分離して管理する

## 運用ルール

- アプリごとに `docs/<app-id>/privacy-policy.html` を配置する
- 一覧ページは `docs/index.html` で管理する

## 公開先

- GitHub Pages: <https://thinKsteM.github.io/thinkstem-legal/>
